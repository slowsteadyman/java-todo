# 📝 simple todo app

---

- 터미널 기반으로 작동하는 간단한 todo app
- 프리코스 기간 동안 배웠던 개발 방법론을 통해 투두 앱을 만들고 지속적으로 개선시킨다
- 데이터베이스를 통해 todo 목록을 관리한다

## ✅ 구현할 기능 목록

---

- sqlite db 연동
- 기본 환경 구성
  - todo 테이블 생성
    - ID / 이름 / 세부내용 / 탭ID / 마감기한
  - tabs 테이블 생성
    - ID / 이름
  - 기본 탭 데이터 삽입
- 할일 추가
- 할일 보기
- 할일 수정
- 할일 삭제
- 터미널 UI 구성
